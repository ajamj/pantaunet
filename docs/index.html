Features

Everything you need to monitor your internet usage

Real-Time Speed

View your current download and upload speeds in real-time with numeric display in the UI.

Per-App Monitoring

See exactly which applications are using your internet and how much data they've transferred.

Usage Alerts

Get notified when your internet usage exceeds configurable thresholds.

Speed History

Visualize your network usage over time with interactive charts.

Dark/Light Theme

Choose between dark and light themes for comfortable monitoring any time of day.

System Tray

Run in the background with system tray integration and quick access menu.

Installation

Get started in minutes

🪟 Windows

  1. Download the .exe installer
  2. Run the installer and follow the prompts
  3. Launch from Start Menu or system tray

🐧 Linux

  1. Download the .AppImage
  2. Make it executable: chmod +x Pantaunet-*.AppImage
  3. Run: ./Pantaunet-*.AppImage

🍎 macOS

  1. Download the .dmg
  2. Open the DMG and drag Pantaunet to Applications
  3. Launch from Launchpad

Build from Source

git clone https://github.com/ajamj/pantaunet.git
cd pantaunet
npm install
npm run tauri build

Requires Node.js 18+, Rust 1.70+, and npm.
