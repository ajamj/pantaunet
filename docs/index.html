Pantaunet is a cross-platform desktop application that helps you understand which applications are consuming your internet bandwidth. Real-time monitoring, speed display, and usage alerts.

Download for Windows View on GitHub

Features

Everything you need to monitor your internet usage

Real-Time Speed

View your current download and upload speeds in real-time with numeric display in the UI.

Per-App Monitoring

See exactly which applications are using your internet and how much data they've transferred.

Usage Alerts

Get notified when your internet usage exceeds configurable thresholds.

Speed History

Visualize your network usage over time with interactive charts.

Dark/Light Theme

Choose between dark and light themes for comfortable monitoring any time of day.

System Tray

Run in the background with system tray integration and quick access menu.
